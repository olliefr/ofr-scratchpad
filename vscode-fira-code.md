# Fira Code settings for VSCode

In `settings.json`,

```json
  "editor.fontFamily": "Fira Code",
  "editor.fontLigatures": "'ss02', 'ss03', 'ss04', 'ss05', 'ss06', 'ss07', 'ss08', 'zero'",
  "editor.fontWeight": 500
```
