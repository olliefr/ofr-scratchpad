# Serve Markdown as HTML with live reload

Useful for viewing a large collection of Markdown files in the browser: [markserv](https://www.npmjs.com/package/markserv)

## Installation

```Shell
$ npm i -g markserv
```

## Usage

```Shell
# Open file 
$ markserv README.md
 
# Directory 
$ markserv ./
```
